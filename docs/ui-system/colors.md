## Regras de uso

1. Studio: verde primario.
2. Domicilio: cor secundaria (`--color-dom`).
3. Acao positiva: verde; destrutiva: danger.
4. Header de modal/tela: `wl-sheet-header-surface`.
5. Header de card: `wl-surface-card-header`.
