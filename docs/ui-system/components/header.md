# Headers (App + Module)

Componentes:
- `AppHeader` (formulários e telas detalhadas)
- `ModuleHeader` (telas principais dos módulos: Agenda, Clientes, Caixa, Menu)

Regras:
- Sticky, fundo branco, `shadow-soft`, `safe-top`.
- `ModuleHeader` começa no topo da tela (sem gap) e não usa cantos arredondados.
- Título com Playfair + texto UI em Lato.
