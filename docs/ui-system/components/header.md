# Header (V2)

Contratos canonicos:

1. Header de tela (Agenda/Novo/Atendimento etc.)
2. Header de modal (bottom sheet)
3. Header de card tipo 1 e tipo 2

Referencias:

1. `docs/ui-system/v2-component-surface-standards.md`
2. classes: `.wl-sheet-header-surface`, `.wl-header-icon-button-strong`

## Regras

1. Header principal usa superficie forte (`wl-sheet-header-surface`).
2. Botoes de icone em header usam padrao forte (`wl-header-icon-button-strong`).
3. Tabs/sheet de modulo ficam na segunda linha do header quando aplicavel.
4. Nao usar header branco quando o contrato da tela pede header forte.
