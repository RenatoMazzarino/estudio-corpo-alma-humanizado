# Botoes (V2)

## Familias canonicas

1. Primario (acao positiva/confirmacao)
2. Secundario (navegacao/acao neutra)
3. Destrutivo (cancelar/excluir/interromper)
4. Icon button de header (`wl-header-icon-button-strong`)
5. Switch/toggle para blocos accordion

## Regras

1. Acoes positivas em verde.
2. Acoes destrutivas em vermelho.
3. Icon buttons de header sempre no padrao de contraste branco + icone verde.
4. Nao criar botao local com radius/cor fora do contrato V2.
