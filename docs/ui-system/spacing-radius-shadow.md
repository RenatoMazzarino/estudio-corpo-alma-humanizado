# Espaçamento, raio e sombra

Espaçamento padrão:
- Header: `px-6` + `safe-top` + `pb-4`
- Cards: `p-4`
- Seções: `mb-6`
- Form: `space-y-4`

Raios:
- Superfícies principais: `rounded-3xl`
- Inputs/cards menores: `rounded-2xl`
- Chips: `rounded-full`

Sombras:
- `shadow-soft` para cards/headers
- `shadow-float` para FAB/overlays
